***AUTO MINIFY YOUR JAVASCRIPT***

1.You must install Microsoft Ajax Minifier 4.0
http://aspnet.codeplex.com/releases/view/40584

2.Your original code must named like xxx.src.js

For Example:
-------------------------------------------------
input js file		|	out put file	|
-------------------------------------------------
HelloWorld.src.js	|	HelloWorld.js	|
default.src.js		|	default.js	|
autominify.src.js	|	autominify.js	|
			|			|
-------------------------------------------------

output js file will be in the same folder
